## Sass Variables for Brand Colors

A SASS module containing variables of all the colors found on [Brand Colors](http://brandcolors.net/ "Brand Colors").

### Usage
1. Include the module:

    ```
    @include "brandcolors";
    ```
2. Use variables:

    ```
    .twitter-button {
        background: $brandcolors-Twitter;
    }
    ```
